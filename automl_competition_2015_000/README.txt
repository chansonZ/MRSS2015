This .zip file contains the submission of the team aad_freiburg for the first AutoML phase of the 2015 automl challenge.

We removed all parts of the code which are specific to the Tweakathon.

The code code has the following dependencies and runs if these are added to the lib folder:

* CMA: https://www.lri.fr/~hansen/cma.py
* Java runtime environment 1.8.0_25: http://www.oracle.com/technetwork/java/javase/downloads/java-archive-javase8-2177648.html#jre-8u25-oth-JPR
* lockfile: https://pypi.python.org/packages/source/l/lockfile/lockfile-0.10.2.tar.gz#md5=1aa6175a6d57f082cd12e7ac6102ab15
* liac-arff: https://github.com/renatopp/liac-arff
* runsolver: http://www.cril.univ-artois.fr/~roussel/runsolver/runsolver-3.3.4.tar.bz2

If there is anything missing, please mail to feurerm@informatik.uni-freiburg.de.
